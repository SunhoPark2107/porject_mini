선정 배경 - 최근, 코로나로 인해 호텔 산업 직격타
취소가 빈발하는 만큼, 예약 취소를 예측하여 이를 대비할 수 있도록 해야 함.

기대 효과 : 취소 예측을 통해, 대비책 타깃을 특정함으로써 호텔 경영 이익 증대

=> 그래서 이 캔슬과 노쇼에 대비하기 위해서 우리가 이렇게 예측을 해보려고 하는데~

모델 소개 => 우리는 이런 특성들을 선택해서, 모델을 고안해 보았다. 모델링은 다음과 같은 기법들을 선택하였고, 
모델 성능을 높이기 위해 이러이러한 방법들을 선택해 보았다.
그리고 최종적으로 선택한 모델은 일반화도 잘 되고 성능도 어느 정도 나오는 이 모델을 선택하였다.


해결책 제시 => 이렇게 우리가 약 80퍼센트 확률로 예약 취소 고객을 분류해내는 모델을 고안해 내었다. 이 모델을 활용하여, 내년부터는 이런 고객들에 대하여 다음과 같은 해결책을 적용해 볼 수 있겠다. 

1. 예약 취소 확률이 높은 고객에게 예약 확인 연락을 더 자주 넣기.
2. 예약 취소 확률이 높은 고객에게 재방문 할인 쿠폰 등 제시하기.
3. 예약 취소를 막기 위해, 예약 취소 확률이 높은 고객에게 호텔 무료 조식 등의 이벤트 제시하기.


기타 => 여행사 별로는 이렇게 취소율이 높은 여행사를 골라 봤다. 머신러닝 상에는 데이터 타입 문제를 해결하지 못해 모델링에 넣지 못하였는데, 이런 식으로 취소율이 높은 여행사에 대해서는 패널티를 준다던가, 예치금 비율을 높인다던가 하여 취소에 대비할 수 있겠다.
